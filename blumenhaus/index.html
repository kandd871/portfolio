See it Blooming Live! ↗

Blumenhaus Magazine is an annual botanical magazine from France. It focuses just as much on the aesthetic side as it does on its written content – described as "a luxurious hybrid of an art book & a magazine." I redesigned/enhanced the logo to evolve with time, reflecting everchanging nature. Using code to manipulate & visualize live data connected to two APIs – Garden API & Weather API – different illustrated flowers bloom every day according to the current season, humidity, & wind speed.
